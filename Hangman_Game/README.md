# 🎯 Hangman Game (Python)

A simple command-line implementation of the classic **Hangman** game built using Python. Guess the hidden word one letter at a time before you run out of lives!

---

## 📌 Features

- 🎮 Classic Hangman gameplay
- 📝 Random word selection from a predefined word list
- ❤️ 6 lives with ASCII Hangman stages
- 🔁 Prevents duplicate guesses
- 🏆 Win/Lose detection
- 🎨 ASCII art logo and hangman illustrations


```bash

```


---

## 🎮 Gameplay

- A random word is selected.
- Guess one letter at a time.
- Correct guesses reveal the letters.
- Wrong guesses reduce your lives.
- Repeated guesses are detected.
- You win by guessing the complete word before your lives reach zero.
