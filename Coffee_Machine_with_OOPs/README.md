# ☕ Coffee Machine (OOP)

A command-line Coffee Machine application built in Python using **Object-Oriented Programming (OOP)** principles. The program allows users to order different types of coffee, processes payments, manages available resources, and prepares drinks accordingly.

---

## 📌 Features

* ☕ Choose from Espresso, Latte, or Cappuccino
* 💰 Accepts coin-based payments
* 📦 Tracks water, milk, and coffee resources
* 📊 Displays machine resource and profit reports
* 🚫 Prevents orders when resources are insufficient
* 🛑 Supports turning the machine off with a command
* 🧩 Built using multiple classes to demonstrate OOP concepts

---

## 🛠️ Technologies Used

* Python 3
* Object-Oriented Programming (OOP)
