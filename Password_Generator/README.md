# 🔐 Password Generator

A simple Python project that generates a secure and random password based on the number of letters, symbols, and numbers selected by the user.

## 📌 Features

- Generate custom-length passwords
- User chooses the number of:
  - Letters
  - Symbols
  - Numbers
- Randomly shuffles all characters for better security
- Beginner-friendly Python project

## 🛠️ Technologies Used

- Python 3
- Random Module
