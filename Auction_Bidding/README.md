# 🏆 Silent Auction

A simple command-line Python application that simulates a silent auction. Participants can enter their names and bids, and the program determines the highest bidder once all bids have been submitted.

---

## 📌 Features

- Accepts multiple bidders
- Stores bids using a Python dictionary
- Automatically determines the highest bidder
- User-friendly command-line interface
- Uses functions for clean and modular code

---

## 🛠️ Technologies Used

- Python 3


2. Navigate to the project folder:

```bash
cd Auction_Bidding
```

3. Run the program:

```bash
python Blind_Auction_Bidding.py
```

---

## 📖 How It Works

1. The program welcomes users to the Silent Auction.
2. Each participant enters:
   - Their name
   - Their bid amount
3. The program asks if there are more bidders.
4. Once all bids are entered, it identifies and displays the highest bidder.
