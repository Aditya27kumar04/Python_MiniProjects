# 🎮 Higher Lower Game

A command-line Python game inspired by the popular **Higher Lower** challenge.

## Features

- Compare two famous personalities
- Guess who has more followers
- Score tracking
- Randomized gameplay
- Modular code using functions

## Technologies

- Python 3
- Random Module

## How to Run

```bash
python higher_lower_main.py
```

## Concepts Practiced

- Functions
- Dictionaries
- Loops
- Conditional Statements
- User Input
- Random Module
